---
title: "Dynamic properties"
description: "Modify animation colors, positions, and values at runtime"
---

Lottie lets you override nearly any animation property at runtime by combining an `AnimationKeypath` with a value provider. Changes take effect immediately — no need to reload the animation.

## Overview

<Steps>
  <Step title="Identify the keypath">
    Find the layer and property name in the animation file. You can use `animationView.logHierarchyKeypaths()` to print all available paths to the console.
  </Step>
  <Step title="Create a value provider">
    Choose the provider that matches the property type (color, float, point, etc.) and initialize it with the value you want.
  </Step>
  <Step title="Register the provider">
    Call `setValueProvider(_:keypath:)` on the `LottieAnimationView`. The animation updates immediately.
  </Step>
</Steps>

## AnimationKeypath

`AnimationKeypath` describes a path through the animation's layer hierarchy using dot-separated component names.

```swift
// Exact path to a specific color node
let keypath = AnimationKeypath(keypath: "Rocket.Body.Fill 1.Color")

// Use string literal syntax (AnimationKeypath is ExpressibleByStringLiteral)
let keypath: AnimationKeypath = "Rocket.Body.Fill 1.Color"

// Build from an array of keys
let keypath = AnimationKeypath(keys: ["Rocket", "Body", "Fill 1", "Color"])
```

### Wildcards

| Wildcard | Behavior |
|---|---|
| `*` | Matches any single layer at the current depth |
| `**` | Matches any layer at any depth |

```swift
// Match "Fill 1" in every layer
let keypath: AnimationKeypath = "**.Fill 1.Color"

// Match every fill color one level inside "Rocket"
let keypath: AnimationKeypath = "Rocket.*.Fill 1.Color"
```

<Tip>
Call `animationView.logHierarchyKeypaths()` during development to discover the exact names of layers and properties in your animation.
</Tip>

## setValueProvider

```swift
public func setValueProvider(_ valueProvider: AnyValueProvider, keypath: AnimationKeypath)
```

Registers a value provider for every property that matches the keypath. Calling this method again with the same keypath replaces the previous provider.

```swift
let keypath = AnimationKeypath(keypath: "**.Fill 1.Color")
let provider = ColorValueProvider(LottieColor(r: 1, g: 0, b: 0, a: 1))
animationView.setValueProvider(provider, keypath: keypath)
```

To remove a previously registered provider:

```swift
animationView.removeValueProvider(for: keypath)
```

## Value providers

### ColorValueProvider

Overrides a color property with a static `LottieColor`, a per-frame closure, or a set of keyframes.

```swift
// Static red color
let provider = ColorValueProvider(LottieColor(r: 1, g: 0.2, b: 0.3, a: 1))

// Per-frame closure (Main Thread rendering engine only)
let provider = ColorValueProvider { frame in
  let progress = frame / 60.0  // assume 60-frame animation
  return LottieColor(r: progress, g: 0, b: 1 - progress, a: 1)
}

// Keyframes
let keyframes = [
  Keyframe(value: LottieColor(r: 1, g: 0, b: 0, a: 1), time: 0),
  Keyframe(value: LottieColor(r: 0, g: 0, b: 1, a: 1), time: 30),
]
let provider = ColorValueProvider(keyframes)

animationView.setValueProvider(
  provider,
  keypath: AnimationKeypath(keypath: "**.Fill 1.Color")
)
```

You can update the color at any time by mutating `provider.color`:

```swift
provider.color = LottieColor(r: 0, g: 1, b: 0, a: 1)
```

### FloatValueProvider

Overrides a scalar numeric property such as opacity or stroke width.

```swift
// Static value — sets opacity to 50%
let provider = FloatValueProvider(0.5)

// Per-frame closure
let provider = FloatValueProvider { frame in
  // Pulse between 0 and 1
  return abs(sin(Double(frame) / 10.0))
}

animationView.setValueProvider(
  provider,
  keypath: AnimationKeypath(keypath: "My Layer.Fill 1.Opacity")
)

// Update the value later
provider.float = 0.8
```

### GradientValueProvider

Overrides a gradient fill with an array of `LottieColor` values and optional stop locations.

```swift
// Two-stop gradient: red to blue
let provider = GradientValueProvider(
  [
    LottieColor(r: 1, g: 0, b: 0, a: 1),
    LottieColor(r: 0, g: 0, b: 1, a: 1),
  ]
)

// Custom stop locations (values between 0.0 and 1.0)
let provider = GradientValueProvider(
  [
    LottieColor(r: 1, g: 0, b: 0, a: 1),
    LottieColor(r: 0, g: 1, b: 0, a: 1),
    LottieColor(r: 0, g: 0, b: 1, a: 1),
  ],
  locations: [0.0, 0.3, 1.0]
)

animationView.setValueProvider(
  provider,
  keypath: AnimationKeypath(keypath: "Background.Gradient Fill 1.Colors")
)
```

### PointValueProvider

Overrides a position or anchor point property.

```swift
// Static point
let provider = PointValueProvider(CGPoint(x: 100, y: 200))

// Per-frame closure
let provider = PointValueProvider { frame in
  CGPoint(x: Double(frame) * 2, y: 100)
}

animationView.setValueProvider(
  provider,
  keypath: AnimationKeypath(keypath: "Layer 1.Transform.Position")
)

// Update later
provider.point = CGPoint(x: 50, y: 50)
```

### SizeValueProvider

Overrides a size property.

```swift
// Static size
let provider = SizeValueProvider(CGSize(width: 200, height: 100))

// Per-frame closure
let provider = SizeValueProvider { frame in
  let scale = 1.0 + (frame / 60.0)
  return CGSize(width: 100 * scale, height: 100 * scale)
}

animationView.setValueProvider(
  provider,
  keypath: AnimationKeypath(keypath: "Icon.Size")
)
```

## Complete example

<CodeGroup>

```swift Change all fill colors
// Make every fill in the animation red
let keypath = AnimationKeypath(keypath: "**.Fill 1.Color")
let redProvider = ColorValueProvider(LottieColor(r: 1, g: 0, b: 0, a: 1))
animationView.setValueProvider(redProvider, keypath: keypath)
```

```swift Change opacity
// Set a specific layer's opacity to 25%
let keypath = AnimationKeypath(keypath: "Overlay Layer.Transform.Opacity")
let opacityProvider = FloatValueProvider(25) // Lottie expects 0–100 for opacity
animationView.setValueProvider(opacityProvider, keypath: keypath)
```

```swift Animate a color with a closure
// Shift hue based on the current frame
let keypath: AnimationKeypath = "**.Stroke 1.Color"
let provider = ColorValueProvider { frame in
  let hue = Double(frame).truncatingRemainder(dividingBy: 60) / 60
  return LottieColor(r: hue, g: 1 - hue, b: 0.5, a: 1)
}
animationView.setValueProvider(provider, keypath: keypath)
```

</CodeGroup>

<Warning>
Per-frame closures (the `block:` initializers) are only supported by the Main Thread rendering engine. When using the Core Animation rendering engine, prefer the static value or keyframe initializers.
</Warning>
