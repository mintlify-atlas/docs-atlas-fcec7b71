Lottie is a cross-platform library that renders animations exported from After Effects as Bodymovin JSON — natively, in real time, with no video files or heavy assets. Drop in a `.json` or `.lottie` file and get a fully interactive, scalable animation in your app.

## Why Lottie?

Lottie lets designers ship animations directly to production without an engineer painstakingly recreating them by hand. Because animations are backed by compact JSON, they are small in size but can represent complex motion.

## Get started in 3 steps

<Steps>
  <Step title="Install Lottie">
    Add the package to your project using Swift Package Manager, CocoaPods, or Carthage. See the [Installation guide](/installation) for detailed instructions.
  </Step>
  <Step title="Add an animation file">
    Export an animation from After Effects using the Bodymovin plugin (or download a free animation from LottieFiles) and add the `.json` or `.lottie` file to your Xcode project.
  </Step>
  <Step title="Display the animation">
    Create a `LottieAnimationView` (UIKit) or `LottieView` (SwiftUI) and point it at your animation file. See the [Quick Start guide](/quickstart) for a working example.
  </Step>
</Steps>
