---
title: "Image, text & font providers"
description: "Customize how Lottie resolves images, text layers, and fonts"
---

Lottie delegates image loading, text substitution, and font resolution to provider objects. Swapping out the default provider lets you load assets from a remote URL, apply branding, or inject user-generated content into an animation at runtime.

---

## Image providers

Some animations reference external image assets. The `AnimationImageProvider` protocol defines how those images are loaded and supplied to `LottieAnimationView`.

### AnimationImageProvider protocol

```swift
public protocol AnimationImageProvider {
    var cacheEligible: Bool { get }
    func imageForAsset(asset: ImageAsset) -> CGImage?
    func contentsGravity(for asset: ImageAsset) -> CALayerContentsGravity
}
```

<ResponseField name="cacheEligible" type="Bool" default="true">
  Whether Lottie may internally cache the result of `imageForAsset`. Set to `false` for providers that return different images at different times (e.g. a remote URL provider that can be refreshed).
</ResponseField>

<ResponseField name="imageForAsset(asset:)" type="(ImageAsset) -> CGImage?">
  Return the image for the given asset descriptor, or `nil` if the image cannot be found. `ImageAsset` carries the asset `name` and `directory` as defined in the animation JSON.
</ResponseField>

<ResponseField name="contentsGravity(for:)" type="(ImageAsset) -> CALayerContentsGravity" default=".resize">
  Controls how the image is positioned within the layer bounds. Defaults to `.resize` (stretch to fill), matching the default `CALayer` behaviour.
</ResponseField>

### Setting an image provider

```swift
animationView.imageProvider = BundleImageProvider(bundle: .main, searchPath: nil)
```

### BundleImageProvider

Loads images by name from a `Bundle`. This is the default provider used when you load an animation with `LottieAnimation.named(_:)`.

<Note>
`BundleImageProvider` is only available on platforms that import UIKit.
</Note>

#### Initialiser

<ParamField path="bundle" type="Bundle" required>
  The bundle to search for image assets.
</ParamField>

<ParamField path="searchPath" type="String?" required>
  An optional subdirectory within the bundle to search first. Pass `nil` to search the bundle root.
</ParamField>

<ParamField path="contentsGravity" type="CALayerContentsGravity" default=".resize">
  How the loaded image is scaled within its layer.
</ParamField>

```swift
// Images alongside the animation in the main bundle
animationView.imageProvider = BundleImageProvider(
    bundle: .main,
    searchPath: nil
)

// Images in a "Animations/Images" subdirectory of the main bundle
animationView.imageProvider = BundleImageProvider(
    bundle: .main,
    searchPath: "Animations/Images"
)

// Images in a separate framework bundle
animationView.imageProvider = BundleImageProvider(
    bundle: Bundle(for: MyFramework.self),
    searchPath: nil
)
```

**Search order** when a `searchPath` is provided:

1. `searchPath/asset.directory/asset.name`
2. `searchPath/asset.name`
3. Bundle root `asset.name`

### FilepathImageProvider

Loads images from an absolute file-system path or `URL`. Useful when images live outside of any bundle, for example in a downloaded cache directory.

<Note>
`FilepathImageProvider` is only available on platforms that import UIKit.
</Note>

#### Initialisers

<ParamField path="filepath" type="String" required>
  The absolute filesystem path to the directory containing image assets.
</ParamField>

<ParamField path="contentsGravity" type="CALayerContentsGravity" default=".resize">
  How the loaded image is scaled within its layer.
</ParamField>

```swift
// Initialise with a String path
let cacheDir = NSSearchPathForDirectoriesInDomains(.cachesDirectory, .userDomainMask, true).first!
animationView.imageProvider = FilepathImageProvider(filepath: cacheDir + "/animation_images")
```

---

<ParamField path="filepath" type="URL" required>
  The directory `URL` containing image assets.
</ParamField>

<ParamField path="contentsGravity" type="CALayerContentsGravity" default=".resize">
  How the loaded image is scaled within its layer.
</ParamField>

```swift
// Initialise with a URL
let imagesURL = FileManager.default.urls(for: .documentDirectory, in: .userDomainMask)[0]
    .appendingPathComponent("lottie_images")
animationView.imageProvider = FilepathImageProvider(filepath: imagesURL)
```

**Search order:**

1. `filepath/asset.name` (direct path)
2. `filepath/asset.directory/asset.name` (with subdirectory)

### Custom image provider

Implement `AnimationImageProvider` to load images from any source, such as a network cache:

```swift
import Lottie
import UIKit

final class RemoteImageProvider: AnimationImageProvider {

    // Don't cache — images may change
    var cacheEligible: Bool { false }

    func imageForAsset(asset: ImageAsset) -> CGImage? {
        // Check your in-memory or disk cache for this asset
        return MyImageCache.shared.image(named: asset.name)?.cgImage
    }

    func contentsGravity(for asset: ImageAsset) -> CALayerContentsGravity {
        .resizeAspect
    }
}

// Usage
animationView.imageProvider = RemoteImageProvider()
```

## Text providers

Text providers let you substitute the static text baked into animation text layers with dynamic values at runtime.

### AnimationKeypathTextProvider protocol

The preferred protocol for text substitution. Receives the full `AnimationKeypath` of the text layer, giving you precise control.

```swift
public protocol AnimationKeypathTextProvider: AnyObject {
    func text(for keypath: AnimationKeypath, sourceText: String) -> String?
}
```

<ResponseField name="text(for:sourceText:)" type="(AnimationKeypath, String) -> String?">
  Return a replacement string for the given layer keypath, or `nil` to keep the original text from the animation file. The `sourceText` argument is the text currently embedded in the JSON for that layer.
</ResponseField>

### DictionaryTextProvider

A built-in implementation that maps layer keypath strings to replacement text. Keys can be either the full keypath string (e.g. `"MY_LAYER.text_value"`) or just the final path component (e.g. `"text_value"`).

#### Initialiser

<ParamField path="values" type="[String: String]" required>
  A dictionary mapping keypath strings (full or last component) to replacement text.
</ParamField>

```swift
let textProvider = DictionaryTextProvider([
    "greeting_label.text_value": "Hello, \(user.firstName)!",
    "subtitle_text": "Welcome back",
])
animationView.textProvider = textProvider
```

<Tip>
If you use the full keypath as a key it takes precedence over a matching last-component key.
</Tip>

### DefaultTextProvider

The default text provider. Returns the `sourceText` unchanged for every layer — the animation plays exactly as authored.

```swift
animationView.textProvider = DefaultTextProvider()
```

### Custom text provider

Implement `AnimationKeypathTextProvider` for runtime logic such as localisation or user-specific content:

```swift
import Lottie

final class LocalizingTextProvider: AnimationKeypathTextProvider {
    func text(for keypath: AnimationKeypath, sourceText: String) -> String? {
        // Use the last keypath component as a localization key
        guard let key = keypath.keys.last else { return nil }
        let localized = NSLocalizedString(key, comment: "")
        // Return nil when there's no localized override to keep the original
        return localized == key ? nil : localized
    }
}

animationView.textProvider = LocalizingTextProvider()
```

### TextContentsScaleProvider

An optional refinement of `AnimationKeypathTextProvider` that also lets you control the `contentsScale` of each text layer. Useful for sharp rendering at non-standard display scales.

```swift
public protocol TextContentsScaleProvider: AnimationKeypathTextProvider {
    func contentsScale(for keypath: AnimationKeypath) -> CGFloat?
}
```

<ResponseField name="contentsScale(for:)" type="(AnimationKeypath) -> CGFloat?">
  Return the desired `contentsScale` for the text layer at the given keypath, or `nil` to use the default.
</ResponseField>

## Font providers

Font providers let you supply custom `CTFont` instances for any text layer in the animation, replacing the family name and size specified in the JSON.

### AnimationFontProvider protocol

```swift
public protocol AnimationFontProvider {
    func fontFor(family: String, size: CGFloat) -> CTFont?
}
```

<ResponseField name="fontFor(family:size:)" type="(String, CGFloat) -> CTFont?">
  Return a `CTFont` to use for text rendered with the given family name and point size, or `nil` to fall back to the system default for that family. The `family` argument is the font family name as specified in the After Effects composition.
</ResponseField>

### DefaultFontProvider

The default font provider. Resolves fonts using `CTFontCreateWithName`, which matches the family name against fonts installed on the device.

```swift
animationView.fontProvider = DefaultFontProvider()
```

### Custom font provider

Implement `AnimationFontProvider` to use custom or downloaded fonts:

```swift
import Lottie
import CoreText

final class BrandFontProvider: AnimationFontProvider {

    func fontFor(family: String, size: CGFloat) -> CTFont? {
        // Override specific families with brand fonts
        switch family {
        case "Helvetica", "Arial":
            // Replace with your custom registered font
            return CTFontCreateWithName("MyBrandSans-Regular" as CFString, size, nil)
        default:
            // Fall through to system font resolution
            return CTFontCreateWithName(family as CFString, size, nil)
        }
    }
}

animationView.fontProvider = BrandFontProvider()
```

<Tip>
Register custom fonts in your app's `Info.plist` under `UIAppFonts` (iOS) or `ATSApplicationFontsPath` (macOS) before assigning a font provider. Core Text can only resolve fonts that are registered with the system.
</Tip>
